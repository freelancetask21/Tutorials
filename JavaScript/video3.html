Basic of variables and constants in JavaScript

In JavaScript we are going to define the variables using keyword let, the only difference between the variables and constants are variables are modify but not constants. variables are containers for storing the information. 

                        let name="thulasi";
                        let score='103';
                        let bonous='20';
                        let totalscore= bonous+score;
                        console.log(name);
                        console.log(totalscore);
                        
                        //
                        let name1="thulasi";
                        let score1=100;
                        let bonous1=20;
                        let totalscore1=score1+bonous1;
                        console.log(name1);
                        console.log(totalscore1);
                        //
                        let firstName="thuasi";
                        let secondName="Ram";
                        let fullname=firstName+ "**" +secondName;
                        console.log(firstName+secondName); // thuasiRam
                        console.log(firstName+" "+secondName); // thuasi Ram
                        console.log(firstName+"_"+secondName);//thuasi_Ram
                        console.log(fullname);//thuasi**Ram
